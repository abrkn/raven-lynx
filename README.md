Automagically report errors and heartbeats

## Installing

`npm install raven-lynx`

## Usage

Add `require('raven-lynx');` to your project and run with the environment variables `SENTRY_DSN` and/or `LYNX_HOST` set.

## Debugging

`DEBUG=raven-lynx npm start`
